# IshaZone- Responsive E-Commerce Web Application

## Overview
This is a responsive e-commerce website.  
The project focuses on clean UI design, proper layout structure, and responsiveness using core web technologies.

---

## Pages
- Home
- Product Listing
- Product Details
- Cart
- Login / Signup
- Admin Dashboard

---

## Technologies Used
- HTML5
- CSS3
- Bootstrap
- Basic JavaScript

---

## Features
- Responsive design for desktop, tablet, and mobile
- Clean navigation and structured sections
- Admin dashboard with sidebar and cards
- Organized file and folder structure
